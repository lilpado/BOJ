## 이 공식을 알고있니?

해당 문제에서
50점과 100점을 나누는 '이 공식을 알고있니?'

```
result = (a * b) % c;

t = (a % b) * (b % c);
if (t > c) t = t % c;
```
이 때 `t == result` 이다.

어쨌든 `(a * b) % c` 만을 고집하지 말고,
유연하게 식을 바꿔서 대처할 필요가 있다. (미리 %c 를 해주는 등)
